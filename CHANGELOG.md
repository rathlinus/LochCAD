# Changelog

Alle nennenswerten Änderungen an LochCAD werden in dieser Datei dokumentiert.

Das Format basiert auf [Keep a Changelog](https://keepachangelog.com/de/1.1.0/).
Dieses Projekt verwendet [Semantic Versioning](https://semver.org/lang/de/).

---

## [0.3.8] — 2026-02-11

### Hinzugefügt
- Open-Source-Release: README, Lizenz, Contributing Guide, Changelog
- GitHub-Templates für Issues und Pull Requests
- Dokumentation (Benutzerhandbuch, Architektur, API-Referenz)
- GitHub Pages Dokumentationsseite

### Geändert
- Hilfe-Menü um Links zu Dokumentation und GitHub erweitert

---

## [0.3.7] — 2026-01-xx

### Hinzugefügt
- Projektmanager mit Multi-Projekt-Verwaltung
- Projektarchiv-Import/Export (.lochcad-archive)
- Intro-Screen mit Kurzanleitung
- Projekt-Notizen

### Verbessert
- Undo/Redo jetzt view-aware (Schaltplan / Lochraster getrennt)
- Autorouter-Performance optimiert

---

## [0.3.0] — 2025-xx-xx

### Hinzugefügt
- 3D-Vorschau mit Three.js / React Three Fiber
- Bauteil-Editor für eigene Symbole und Footprints
- Auto-Layout für Lochraster
- Autorouter für Lochraster-Verbindungen
- ERC (Electrical Rules Check)
- DRC (Design Rules Check)
- SPICE-Netzliste Export
- Stückliste (BOM) als CSV und HTML
- Multi-Sheet-Support im Schaltplan
- Stripboard-Modus
- Cut-Track-Werkzeug
- Lötbrücken und Drahtbrücken

---

## [0.2.0] — 2025-xx-xx

### Hinzugefügt
- Schaltplan-Editor (Konva-basiert)
- Lochraster-Editor (Konva-basiert)
- Eingebaute Bauteilbibliothek (Widerstände, Kondensatoren, ICs, etc.)
- Drag & Drop Bauteil-Platzierung
- Draht-Werkzeug
- Properties-Panel
- Zoom & Pan
- Projekt speichern/laden (LocalStorage + .lochcad-Datei)

## [0.1.0] — 2025-xx-xx
