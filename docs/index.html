LochCAD

Open-Source Schaltplan- & Lochraster-CAD — direkt im Browser. Schaltpläne zeichnen, auf Lochraster übertragen, in 3D prüfen und exportieren.

App starten GitHub Repository

Features

Alles was du brauchst — vom Schaltplan bis zur fertigen Platine.

SCH

Schaltplan-Editor

Professioneller Schaltplan-Editor mit Symbolen, Drähten, Bussen, Net-Labels und Multi-Sheet-Support.

PCB

Lochraster-Editor

Bauteile auf Perfboard oder Stripboard platzieren. Drähte, Lötbrücken, Drahtbrücken und Track-Cuts.

3D

3D-Vorschau

Interaktive 3D-Ansicht der bestückten Platine — drehen, zoomen, kontrollieren.

LIB

Bauteil-Editor

Eigene Schaltplansymbole und Footprints definieren und in Projekten verwenden.

ART

Autorouter & Auto-Layout

Automatisches Routen und Platzieren — spart Zeit bei komplexen Schaltungen.

DRC

ERC & DRC

Electrical und Design Rules Check finden Fehler, bevor sie auf der Platine landen.

EXP

Export

SPICE-Netzliste, Stückliste (CSV/HTML), Projektdatei — alles mit einem Klick.

OFF

Offline & Privat

Läuft komplett im Browser. Keine Daten verlassen deinen Rechner — kein Account nötig.

Schnellstart

In unter 2 Minuten lokal loslegen.

Installation

# Repository klonen
git clone https://github.com/linusrath/LochCAD.git
cd LochCAD

# Abhängigkeiten installieren
npm install

# Entwicklungsserver starten
npm run dev

# → http://localhost:3000
